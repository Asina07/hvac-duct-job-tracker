# HVAC Material Management System

A full-stack web application built to replace manual Excel tracking 
for a project engineer in the ducting industry.

## Features
- JWT Authentication — secure login system
- Job Management — create, update, track and delete jobs
- Status Tracking — real-time status updates per job
- Material & Project Management
- Delivery Tracking — track delivery lots per job
- Dashboard — live summary of total SQM, pending jobs, 
  delivered quantities grouped by material and status

## Tech Stack
- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL (Supabase)
- Authentication: JWT

## Purpose
Built as a real-world project for practical use, replacing a 
manual Excel-based workflow with a proper web application.
